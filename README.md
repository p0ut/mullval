# Mullval
A small batch script designed to check your ping on Valorant, while tunneling your connection through Mullvad VPN.

# Usage
Quite literally runs through every single server and displays a results table in the end.

<img src="https://i.imgur.com/QkVITga.png" alt="Project Banner" width="100%" />

# Support
As of currently, runs checks on NA Virginia, Chicago, San Jose and Portland. This runs checks on the same AWS nodes assuming the same routing. 
**Please just use this for reference, by no means will this ever be full fledged accurate**
